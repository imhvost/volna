## Theme Building

This is a Gulp build that is compiled from `/wp-content/themes/volna/src/`
Please make edits exactly in the folder `/wp-content/themes/volna/src/`

## Project Setup

```sh
npm install
composer install
```

### Compile for Development

```sh
gulp
```

### Compile for Production

```sh
gulp build
```

### Compile and Minify for Production

```sh
gulp min
```
